# German Credit Risk Project

A structured credit-risk analysis project built with Python, NumPy, Pandas, and Matplotlib.

## Goal
To explore and model credit risk using a real financial dataset, focusing on:
- exploratory data analysis
- statistical testing
- information-theoretic analysis
- logistic regression from scratch
- model evaluation and validation

## Tools
- Python
- NumPy
- Pandas
- Matplotlib
- Scipy

## Project Structure
- `data/raw/` - original dataset
- `data/processed/` - cleaned datasets
- `src/` - source code
- `outputs/figures/` - generated plots
- `outputs/reports/` - reports and summaries
